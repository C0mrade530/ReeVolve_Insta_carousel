# RealPost Pro — Стратегия MVP для продажи риелторам

## Текущий статус: честный аудит

### Работает полноценно:
- Авторизация (Supabase)
- Генератор каруселей (GPT + Pillow) — 8 дизайнов, 8 шрифтов, оценка качества
- Персональный шаблон эксперта (фото → AI-обработка)
- Контент-план (календарь, отображение)
- Конкуренты: ручная вставка текста → анализ → рерайт → карусель
- Дашборд (базовый)

### Заглушки (UI есть, бэк не работает):
- Подключение Instagram аккаунта (логин через instagrapi — TODO)
- Публикация в Instagram (все методы stub)
- Авто-планирование (БД есть, Celery не подключён)
- Аналитика (нет реальных данных из IG)
- Парсер ЦИАН/Авито/Яндекс (return [])
- Парсинг конкурентов автоматом (зависит от IG сессии)

### Полностью отсутствует:
- Биллинг / подписки / оплата
- Лендинг для продажи
- Онбординг нового пользователя

## Стратегия: Что такое $1M капитализация?

**$1M cap** = 100 клиентов × $200/мес × 4+ лет LTV,
или 50 клиентов × $400/мес × 4+ лет.

**Целевая аудитория:** Риелторы и агентства недвижимости, которые:
- Знают что Instagram = клиенты, но не умеют / нет времени вести
- Готовы платить $200-500/мес за автоматизацию
- Публикуют 3-5 постов в неделю (или хотят)

**Ценность продукта (value prop):**
> "Подключи аккаунт → Нажми кнопку → Каждый день у тебя новая экспертная карусель.
> Контент как у топ-блогеров, но без найма SMM-щика за 50K/мес."

## ПЛАН MVP — 5 критических фичей для продажи

### ПРИОРИТЕТ 1: Автопубликация в Instagram (без неё продукт бесполезен)

**Что делаем:**
- Реальный логин через `instagrapi` (username + password → session)
- Публикация карусели (publish_album)
- Проверка и обновление сессии
- Обработка 2FA (challenge resolver есть в instagrapi)

**Почему критично:** Без авто-публикации продукт = "красивая генерилка картинок",
которую никто не купит за $200/мес. С публикацией = "полноценная SMM-автоматизация".

**Время:** 4-6 часов

---

### ПРИОРИТЕТ 2: Расписание + автопостинг (set it and forget it)

**Что делаем:**
- Кнопка "Запланировать на [дата/время]" после генерации
- Фоновый процесс проверяет расписание каждые 5 мин
- Автоматически публикует когда время пришло
- Настраиваемое расписание: Пн-Пт в 10:00, 14:00, 19:00

**Почему критично:** Риелтору нужно нажать кнопку ОДИН РАЗ В НЕДЕЛЮ,
а не сидеть каждый день. "Настрой расписание на месяц за 30 минут."

**Время:** 3-4 часа (Celery beat + простой scheduler)

---

### ПРИОРИТЕТ 3: Биллинг (без оплаты — нет бизнеса)

**Что делаем:**
- Интеграция Юкасса (ЮMoney) или Stripe (для международных)
- 3 тарифа:
  - **Старт** — 2990₽/мес (1 аккаунт, 30 каруселей/мес, базовые дизайны)
  - **Про** — 5990₽/мес (3 аккаунта, безлимит каруселей, все дизайны, конкуренты)
  - **Агентство** — 14990₽/мес (10 аккаунтов, приоритетная генерация, API доступ)
- Триал 7 дней бесплатно (без карты)
- Лимиты по тарифу (генераций/мес, аккаунтов)

**Почему критично:** Без биллинга продукт нельзя монетизировать.

**Время:** 6-8 часов

---

### ПРИОРИТЕТ 4: Лендинг (без него некуда слать трафик)

**Что делаем:**
- Одностраничник на том же React (или отдельный Astro/Next)
- Блоки: Герой → Боль → Решение → Демо (видео/GIF) → Тарифы → FAQ → CTA
- Ключевые триггеры:
  - "Вместо SMM-щика за 50K/мес"
  - "Контент уровня топ-блогеров за 5 минут"
  - "Уже 200+ риелторов автоматизировали Instagram" (социальное доказательство)
  - Живые примеры каруселей (скриншоты реальных генераций)

**Почему критично:** Нет лендинга = нет конверсии. Даже лучший продукт
не продаётся без внятного сайта.

**Время:** 4-6 часов

---

### ПРИОРИТЕТ 5: Онбординг нового пользователя

**Что делаем:**
- Wizard при первом входе: "Подключи Instagram → Загрузи фото → Выбери стиль → Сгенерируй первую карусель"
- 4 шага с прогресс-баром
- Автоматическая первая генерация (WOW-эффект за 60 секунд)
- Подсказки по интерфейсу

**Почему критично:** Retention. Если человек не увидел результат за первые 2 минуты —
он уйдёт и никогда не заплатит.

**Время:** 3-4 часа

## Что НЕ нужно для MVP (отложить):

| Фича | Почему не сейчас |
|-------|------------------|
| Парсер ЦИАН/Авито | Тематические карусели продают лучше объектных. Добавить после 50 клиентов |
| Аналитика из IG | Дорого в разработке, риелторы смотрят метрики в самом Instagram |
| Авто-парсинг конкурентов | Ручная вставка текста работает для MVP, автоматика — v2 |
| Мобильное приложение | Telegram-бот или PWA дешевле и быстрее |
| Мультиязычность | Фокус на РФ/СНГ |
| AI Reels/Video | Отдельный огромный продукт |

## Go-to-Market: Как продавать?

### Каналы:
1. **Telegram-чаты риелторов** — бесплатно сгенерировать 3 карусели, показать результат
2. **Instagram таргет** — на аудиторию "интересы: недвижимость + бизнес"
3. **YouTube/Reels** — скринкасты: "Как я генерирую 30 постов за 15 минут"
4. **Партнёрства** — агентства недвижимости (белый лейбл)
5. **Продуктовые лончи** — Product Hunt (для международной версии)

### Pricing Psychology:
- **Якорь:** "SMM-щик = 50-80K/мес" → наш продукт = 3-6K/мес (в 10x дешевле)
- **Триал:** 7 дней бесплатно, без карты → показать WOW → конверсия в оплату
- **Годовая скидка:** -30% за годовую подписку (cash flow upfront)

### Unit Economics:
- CAC (стоимость привлечения): ~3000₽
- ARPU (средний чек): ~5000₽/мес
- LTV: 5000 × 12 мес = 60000₽
- LTV/CAC = 20x (отличная экономика)

1. **Автопубликация** — продукт из "генерилки" превращается в "автоматизацию"

3. **Биллинг** — можно брать деньги
4. **Лендинг** — есть куда слать трафик

**Всё остальное — после первых 50 клиентов.**
